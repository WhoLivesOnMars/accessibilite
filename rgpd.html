Cette politique explique comment nous collectons et utilisons vos données personnelles conformément au RGPD.

Données collectées

Nous collectons uniquement les informations nécessaires pour fournir nos services : nom, email, messages via le formulaire de contact, et données techniques liées à la navigation sur le site.

Utilisation des données

Les données sont utilisées uniquement pour :

  • Répondre aux demandes des utilisateurs
  • Améliorer le site et ses services
  • Respecter les obligations légales

Partage des données

Nous ne partageons vos données personnelles avec aucun tiers, sauf si la loi l’exige ou pour fournir un service demandé.

Droits des utilisateurs

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Les corriger ou les supprimer
  • Retirer votre consentement au traitement des données

Pour exercer ces droits, contactez-nous à contact@monsite.fr.

Sécurité

Nous mettons en place des mesures techniques et organisationnelles pour protéger vos données contre tout accès non autorisé.

Durée de conservation

Vos données sont conservées uniquement le temps nécessaire aux finalités décrites ci-dessus.
